Basic HTML / CSS Usage:

We'll spend the next 30 mins getting used to VSCode and building a simple web page.

We will create some CSS and CSS classes and then save that code to an external CSS file.

Finally we will link our basic web page to the external CSS.

Doing all of the above will help prepare you for the rest of today.

I will be sharing code with you but you'll need to have some idea where the code goes and how it relates to the page and page style.

Start VSCode Project
  • Create Folder on your machine and give it a meaningful name - e.g. myProject
    Remember where you made the folder.This folder will be your project home.
  • Open VSCode and use "File > Open Folder" and navigate to the folder you just created. Or, drag and drop the new folder into the current VSCode window. Either of the 2 methods will create a new project in VSCode.
  • In VSCode your working files are on the left of the window - "Explorer". Using the explorer create these 3 files.
    1. index.html
    2. shared.css
    3. navigation.css

File names are always Case-Sensitive.
